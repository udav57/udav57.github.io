Как сделать ссылку

Для создания ссылки используется тег <a> с атрибутом href.

Атрибут target

Атрибут target определяет, где откроется ссылка. Например, _blank откроет её в новой вкладке.

Изображение в качестве ссылки

Изображение можно сделать ссылкой, обернув тег <img> в <a>.

Ссылки внутри страницы

Якоря позволяют перемещаться по странице. Например, эта секция доступна по ссылке #page-links.
